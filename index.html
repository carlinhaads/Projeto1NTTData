O que são Clients

(voltar)

É o computador que está do outro lado, que foi configurado para utilizar o serviço que é oferecido pelo servidor.

O nosso computador chamado cliente, irá disponibilizar a partir de um computador servidor. É o que é mostrado pro usuário

O que são servers

Servidor é o computador no qual foi configurado em uma rede para oferecer algum tipo de serviço.

Introdução ao HTML na prática

(voltar)

HTML não é uma linguagem de programação, é um arquivo de marcação, comtextos delimitados por nomes que o navegador consegue interpretar

Cada nome destes delimitadores, tem uma função e comportamento específico
